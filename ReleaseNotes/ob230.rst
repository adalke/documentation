Open Babel 2.3.0
================

Released on the 201x/nn/nn.

New file formats:

* Turbomole AOFORCE
* JSON formats from PubChem and from ChemDoodle
* DALTON input/output
* XXX's extended XYZ format
* LPMD
* MDFF
* ORCAA
* "Painter" format for custom 2D depictions

New features:

* Confab, the systematic conformation generator
