MDL MOL/SDF format (mol, mdl, sdf, sd)
======================================

MDL MOL/SDF format Reads and writes V2000 and V3000 versions

Write Options
~~~~~~~~~~~~~
**3**
    output V3000 not V2000 (used for >999 atoms/bonds)
**m**
    write no properties
**Input**
    Options, e.g. -av
**v**
    AtomValue in V line is NOT a partial charge
